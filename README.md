# VoidTube

VoidTube is the minimalist's choice for a distraction-free YouTube experience. No noise, no clutter—just your focus.

![VoidTube Logo](icon.png)

---

## Why VoidTube?

YouTube's interface is engineered to keep you clicking. Between recommendation walls, ever-present sidebars, and endless Shorts, it's easy to lose track of why you opened the site.

VoidTube restores your digital agency. It doesn't just "block" elements; it creates a focused void that prevents the initial hook from ever landing.

---

## Features

VoidTube gives you granular control over what you see:
*   Home Feed: Hide the addictive recommendation wall.
*   Sidebar: Remove "Up Next" and related videos from the watch page.
*   Comments: Deep-clean the page by hiding the entire comment section.
*   Shorts: Block Shorts shelves across the platform to stay focused on long-form content.
*   Master Toggle: Pause or resume protection with a single click.

---

## Technical Highlight

Unlike other blockers that can cause layout shifts, VoidTube uses a high-performance CSS Injection engine. By running at document_start, the extension hides distracting elements before they are even rendered, ensuring a perfectly smooth, distraction-free load every time.

## Installation

### Firefox (Recommended)
1.  Visit the Firefox Add-on Store.
2.  Search for VoidTube.
3.  Click Add to Firefox.

### Self-Hosting / Development
1.  Clone this repository.
2.  Open Firefox and go to about:debugging.
3.  Click This Firefox then Load Temporary Add-on...
4.  Select the manifest.json file in this directory.
